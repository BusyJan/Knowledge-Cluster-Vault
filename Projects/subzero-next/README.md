## Current state

- Flat single-page schematic for review: `subzero-next-flat.kicad_pro` (generated by `scripts/merge_schematic_flat.py`; see `FLAT-SCHEMATIC.txt`). Primary hierarchical project remains `subzero-next.kicad_pro`.
- **Schematic status (2026-04-18):** ✅ **Electrically clean.** ERC: 0 errors / 6 acceptable warnings (all intentional — see [[Decisions]] 2026-04-18 sign-off entry). Netlist: 207 components, 384 nets, all critical rails are unique single nets.
- **PCB status (2026-04-18):** ✅ **TWO PCBs initialized — sandwich design.** Generated via `scripts/generate_2board.py`:
  - `pcb/subzero-top.kicad_pcb` — 80×130 mm portrait, 2-layer 0.8 mm. **F.Cu** = display + ANO 5-way + BACK/BOOT/RESET buttons + NeoPixel strip + IR TX/RX + status LEDs. **B.Cu** = entire power+USB+charger stack, microSD, NFC/RFID, GPS, IMU, flash, AW9523×2, buzzer, vibration motor, ATECC608B crypto, J_B2B_T (Hirose DF12-50 header). 138 components.
  - `pcb/subzero-main.kicad_pcb` — 80×130 mm portrait, 4-layer 1.6 mm with internal GND + PWR planes. **F.Cu** = ESP32-S3 + ESP32-C6 + nRF52840 + 2× CC1101 + nRF24 + LoRa + 7× SMA antennas (top edge) + ESD diodes + J_B2B_M (Hirose DF12-50 socket). **B.Cu** = intentionally empty (flat surface for the LiPo battery in the sandwich gap). 69 components.
  - **B2B**: Hirose DF12-50, 3.0 mm stack height, mates centrally on both boards. 42 cross-board nets routed through it (power/USB/SPI/I2C/RF-control/MCU-debug). 8 spare pins.
  - **Sandwich stack**: TOP-PCB (0.8) + LiPo (~2 mm Pouch in 3.0 mm gap) + MAIN-PCB (1.6) ≈ 4.4 mm total + B2B housing.
  - Both PCBs are independently orderable at JLCPCB / PCBWay (separate Gerbers/BOMs/CPLs per board).
- **Footprints:** 207/207 assigned (custom Ebyte modules + ATGM336H GPS in `libs/project-apex.pretty/`; passives 0603; ICs per datasheet — see [[Notes]] 2026-04-17 entries).
- **Pending for next revision:** (a) NeoPixel decoupling caps + series resistor (currently NC for prototype), (b) DW01A external N-MOSFETs for actual LiPo protection, (c) GPS active antenna bias-T if needed, (d) thermal monitoring via TH1 if needed. None block prototype validation.
